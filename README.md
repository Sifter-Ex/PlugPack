# Overview

Sifter's PlugPack will install all extentions if sifter is already installed on the system.<br />
This is to allow all plugins to quickly be proccessed and installed without the need to seperately install each.<br />

---------------------------------------------------------------------------------------------------------------------

# Sifter's Tested OS

	Working on: - Kali
		    - Parrot
		    - Ubuntu
		    - Linux (any distro)
		    - Windows (Linux Subsystem with Docker and VcXsrc - for xterm - installed correctly, alternatively you can use KeX)
		    
Works on windows with linux-subsystem but please ensure docker is properly installed and configured, <br /> 
following the instructions from <a href="https://docker.io">docker website</a><br />
nMap doesn't work on windows, due to port issues<br />
but Zenmap (nMap GUI) can be used instead. A few other issues may arise too.<br />
Untested on mac, though theoretically the same should apply to mac as windows - regarding docker install & tools

# Installation:
	
	[!] To install the PlugPack, copy and paste the following into a terminal:
	*
	$ git clone https://github.com/Sifter-Ex/PlugPack; cd PlugPack; bash plugin.sh

    [!] For a single plugin installation, instructions can be found in the corresponding repo
    + cPlug - https://github.com/Sifter-Ex/cPlug
    + gPlug - https://github.com/Sifter-Ex/gPlug
    + mPlug - https://github.com/Sifter-Ex/mPlug
    + fPlug - https://github.com/Sifter-Ex/fPlug
